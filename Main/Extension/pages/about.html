About the Extension

The Anti-Phishing Browser Extension is a real-time security tool designed to detect and block malicious or suspicious URLs before they can harm users. It leverages both rule-based detection and Google Safe Browsing (GSB) API integration to provide an extra layer of protection against phishing, malware, and unsafe domains.

Current Features

  • Real-Time Request Interception - The extension uses the webRequest and webRequestBlocking APIs to monitor every outgoing request from the browser.
  • Custom Domain Blocking by User - Users can manually block requests to specific domains in the “Manage Blocked Domains” page.
  • Integration with Google Safe Browsing (GSB) API - For extra security protection that Google has latest blacklisted Phishing URLs
  • Network Request Logging - The popup displays the recent 200 network activities, helping users analyze browsing behavior and what resources are requested & downloaded.

Permissions used

  • webRequest and webRequestBlocking - To intercept & analyze outgoing requests
  • storage - To maintain user-defined blocklists and network history.
  • tabs - To track the currently active site and update its safety status.
